# Instalation

```bash
npx degit sveltegaruda/theme-material my-project
cd my-project
npm i
```

# Feature

## Code splitting

Execute command below 

```bash
npm run build
```

## File based routed

Using Routify.

## Hot module reload

The state is not reset when we develop the application. Just run:

```bash
npm run dev
```

## Material Design

Material CSS.
